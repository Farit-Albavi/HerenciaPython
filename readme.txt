Instrucciones para probar el sistema de reservas de Python Airlines:

1. Uso del programa:
   - El programa te dará la bienvenida y mostrará la disponibilidad y los precios de los asientos.
   - Puedes elegir entre comprar un asiento o salir del programa.
   - Si eliges comprar un asiento, debes especificar si deseas un asiento "primera" o "economico" y la cantidad de boletos que deseas reservar.
   - El programa verificará la disponibilidad y realizará la reserva si es posible.
   - Puedes continuar comprando asientos o salir del programa en cualquier momento.

2. Resultados:
   - Cuando decidas salir del programa, se calculará la cantidad de asientos comprados por clase, el precio a pagar por cada clase y el precio total de todos los asientos reservados y se mostrará en pantalla.

3. Prueba adicional:
   - Puedes realizar varias pruebas para comprobar la funcionalidad del sistema de reservas con diferentes opciones y cantidades de asientos.
